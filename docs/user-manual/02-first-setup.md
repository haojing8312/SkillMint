## 第一次必做配置
优先使用首次启动出现的“快速配置（1分钟）”向导：
1. 在引导层点击“快速配置（1分钟）”。
2. 选择服务商预设并填写 API Key。
3. 点击“测试连接”。
4. 测试通过后点击“保存并开始”。

如果你跳过了向导，也可以用完整设置路径：
1. 点击左下角“设置”。
2. 进入“模型连接”页（默认打开）。
3. 新增一个模型连接：填写名称、API Key、Base URL、模型名。
4. 点击“测试连接”确认可用。
5. 点击“保存”。

## 推荐模型（截至 2026-03-06）
建议优先选择稳定、文档清晰、生态成熟的模型。

国外推荐：
1. OpenAI：`GPT-5.4`（你提到的名称正确）。

## 可选优化配置
1. 在同一页下半部分配置“语言与沉浸式翻译”。
2. 默认语言建议保持 `zh-CN`（按团队规范可调整）。
3. 翻译策略建议选择“优先模型，失败回退免费翻译（推荐）”。
4. 如果你需要统一默认目录，可在“智能体员工”页设置全局默认工作目录。

## 成功标志
1. “模型连接”保存成功，不再出现缺失 API Key 的报错。
2. 返回“开始任务”后能正常创建会话并收到回复。

## 常见问题
1. 测试连接失败：优先检查 API Key 是否完整、Base URL 是否正确。
2. 保存后仍不可用：重新打开设置，确认该模型确实已保存。
3. 翻译结果异常：先关闭沉浸式翻译确认主流程可用，再回头调翻译策略。
