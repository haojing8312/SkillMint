## 你将获得什么

- 你知道文档里不会出现未上线内容。
- 你知道下一步从哪里开始。

## 当前可用能力（面向普通用户）
1. 在“开始任务”中发起会话并持续追问。
2. 在“智能体员工”中管理员工并与指定员工开始任务。
3. 在“专家技能”中查看技能、安装技能、创建技能、打包技能。
4. 在“设置”中配置模型连接、语言与翻译、搜索引擎、MCP 服务器。

> 截图位置：`S00-01`  
> 建议截图：左侧导航（开始任务 / 专家技能 / 智能体员工 / 设置）。

## 当前不包含能力
1. 企业权限、SSO、多租户、组织级管理能力。
2. 文档未提到的规划中能力。
3. 仅维护者使用的底层集成与调试流程。
