## 阅读顺序
1. 第一次上手：按 0 -> 1 -> 2 -> 3 逐章执行。
2. 需要角色化协作：继续看第 4 章。
3. 需要扩展能力：继续看第 5 章。
4. 需要调优体验：看第 6 章。
5. 遇到异常：先看第 7 章，再联系支持。

## 术语约定
- “开始任务”：左侧主导航入口。
- “专家技能”：左侧主导航入口，包含“我的技能 / 技能库 / 找技能”。
- “智能体员工”：左侧主导航入口，用于管理员工角色与飞书配置。
- “设置”：左下角设置入口。

## 文档边界（严格）
- 本手册不包含未上线能力。
- 本手册不包含企业版能力（多租户、SSO、RBAC 等）。
- 本手册不包含纯开发者维护操作。
