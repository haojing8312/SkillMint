## 说明
本附录用于统一手册中的模型推荐与命名，避免用户在配置时填写不存在的型号。

## 推荐清单（面向普通用户）

1. OpenAI：`GPT-5.4`
2. Anthropic：`Claude 4.6` 代（`Claude Opus 4.6` / `Claude Sonnet 4.6`）

国内：
1. 智谱：`GLM-4.7`
2. MiniMax：`MiniMax-M2.5`
3. Moonshot：`Kimi K2` 系列

2. `claude 4.7` -> 建议改为：`Claude 4.6` 代（当前官方命名）

## 使用建议
1. 新手优先选“一个主力模型 + 一个备用模型”。
2. 同一时期手册内只保留一套推荐名单，按月更新。
3. 每次改推荐名单，必须同步更新第 2 章与第 6 章。
