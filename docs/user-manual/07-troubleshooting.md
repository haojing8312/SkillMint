# 7. 常见问题与排错

## 文档信息
- 目标读者：使用中遇到异常的普通用户
- 预计阅读：10 分钟
- 使用原则：先按本页排查，再反馈

## 目录
1. 快速排查顺序
2. 高发问题清单
3. 技能安装冲突
4. 飞书连接异常
5. 反馈模板

## 快速排查顺序
1. 先看是否是模型连接问题（最常见）。
2. 再看是否是当前页面配置未保存。
3. 再看是否需要重启应用。
4. 最后提交反馈。

## 高发问题清单
### 1) 可以打开应用，但消息发不出去
- 排查：
  1. 进入“设置 -> 模型连接”，执行“测试连接”。
  2. 确认至少有一个可用模型连接。
  3. 返回“开始任务”重试。

### 2) 能发消息，但返回质量不稳定
- 排查：
  1. 把问题改为“目标 + 限制 + 输出格式”。
  2. 在同一会话里追问，不要每次新开会话。

### 3) 安装技能失败
- 排查：
  1. 检查是否选择了正确安装模式。
  2. `.skillpack` 模式确认已填用户名。
  3. ClawHub 模式确认已实际选中一个技能项。

### 4) 员工页飞书状态异常
- 排查：
  1. 检查 App ID / App Secret 是否填写并保存。
  2. 点击“重试连接”后等待几秒再查看状态。

## 技能安装冲突（重名）
当你遇到重名冲突，参考项目现有文档处理：
- `docs/troubleshooting/skill-installation.md`

推荐做法：
1. 给技能名加前缀（组织/个人）。
2. 加版本尾缀区分不同迭代。

## 飞书连接异常
1. 员工是启用状态且已配置 App ID / App Secret。
2. 使用“重试连接”后观察状态点和提示文案。
3. 若仍失败，记录错误文案并提交给维护者。
