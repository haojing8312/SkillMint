## 这页解决什么
你将学会用“员工角色”来稳定输出风格，并管理员工级配置。

## 入口与页面结构
1. 左侧点击“智能体员工”。
2. 页面分为员工列表和员工详情两部分。
3. 页面顶部有“打开员工助手”入口。

> 截图位置：`S04-01`  
> 建议截图：智能体员工页全貌（列表区 + 详情区）。

## 创建员工（当前推荐方式）
当前页面提示已移除手动创建流程，推荐走内置“智能体员工助手”：
1. 点击“打开员工助手”或“新建员工”。
2. 在对话中描述岗位职责和目标。
3. 按助手引导确认配置草案后创建。
4. 创建后返回员工列表，确认新员工可见。

> 截图位置：`S04-02`  
> 建议截图：员工页“打开员工助手 / 新建员工”按钮。

## 调整员工与设为主员工
1. 在列表中选择某个员工。
2. 在详情区点击“调整员工”，进入员工助手做修改。
3. 如需默认进入该员工，点击“设为主员工并进入首页”。
4. 如需直接执行任务，点击“与该员工对话开始任务”。

> 截图位置：`S04-03`  
> 建议截图：详情区按钮（调整员工、设为主员工、开始任务）。

## 飞书连接配置
1. 在员工详情中的“飞书连接与配置”区域填写：
   - open_id（可选）
   - App ID
   - App Secret
2. 点击“保存飞书配置”。
3. 点击“重试连接”触发重连并观察状态点颜色与文案。

> 截图位置：`S04-04`  
> 建议截图：飞书配置表单与“保存飞书配置/重试连接”按钮。

## 长期记忆管理
1. 在“长期记忆管理”区域选择范围（全部技能或某个技能）。
2. 点击“刷新统计”查看文件数和体积。
3. 需要备份时点击“导出 JSON”。
4. 需要重置时点击“清空记忆”（有高风险确认）。

> 截图位置：`S04-05`  
> 建议截图：长期记忆管理区（范围下拉、刷新、导出、清空）。

## 常见问题
1. 找不到手动新建表单：当前版本已改为通过员工助手创建。
2. 飞书状态红色：先检查 App ID / App Secret，再点“重试连接”。
3. 清空记忆后还能看到旧内容：先刷新统计并重新进入会话验证。
