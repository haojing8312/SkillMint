# 附录 A. 截图点位清单

## 使用说明
1. 本清单用于标记每个章节的截图插入位置。
2. 文件建议统一放在：`docs/screenshots/user-manual/`。
3. 命名建议：`S章节-序号-页面名.png`。
4. 每张图只标一个重点，避免多重标注。

## 点位清单
- `S00-01`：左侧导航总览（开始任务 / 专家技能 / 智能体员工 / 设置）
- `S01-01`：安装完成后首次打开主界面
- `S01-02`：首次启动快速配置引导（Gate 或提示条）
- `S01-03`：完成配置后的主界面验收点（导航 + 设置）
- `S02-01`：快速配置弹窗（测试连接 / 保存并开始）
- `S02-02`：模型连接表单（测试连接 / 保存）
- `S02-03`：语言与沉浸式翻译配置
- `S03-01`：开始任务页输入框与开始按钮
- `S03-02`：聊天页发送按钮与回复区域
- `S04-01`：智能体员工页面总览
- `S04-02`：打开员工助手/新建员工入口
- `S04-03`：员工详情区关键按钮
- `S04-04`：飞书连接与配置区域
- `S04-05`：长期记忆管理区域
- `S05-01`：专家技能页总览（按钮+Tab）
- `S05-02`：我的技能卡片操作区
- `S05-03`：技能库/找技能页面
- `S05-04`：安装 Skill 弹窗模式切换
- `S05-05`：`.skillpack` 安装模式表单
- `S05-06`：技能打包页（选择目录/导出技能包）
- `S06-01`：设置页主要分区
- `S06-02`：模型连接配置区
- `S06-03`：语言与翻译配置区
- `S06-04`：搜索引擎配置区
- `S06-05`：MCP 服务器配置区

## 飞书发布建议
1. 飞书正文中先放步骤，再放对应截图。
2. 每张截图上方写“你应该看到什么”。
3. 每章至少 2 张关键图，避免全是文字。
