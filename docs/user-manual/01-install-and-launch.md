# 1. 安装与启动（Windows）

## 文档信息
- 目标读者：第一次安装用户
- 预计耗时：5-10 分钟
- 成功标准：首次启动看到“快速配置（1分钟）”向导或提示

## 这页解决什么
把 WorkClaw 正常安装并启动到可操作状态。

## 开始前准备
1. Windows 系统可联网。
2. 你已获得安装包（内部发布或项目发布页）。

## 安装步骤
1. 双击安装包并按向导完成安装。
2. 安装结束后，从桌面或开始菜单打开 WorkClaw。
3. 首次打开时，等待主界面渲染完成，不要连续重复点击。

> 截图位置：`S01-01`  
> 建议截图：安装完成后的应用主窗口。

## 启动与验收
首次安装后的第一次启动，默认会进入模型快速配置引导（无模型时）：
1. 看到“首次使用需要先连接一个大模型”提示层或“快速配置（1分钟）”提示条。
2. 可点击“快速配置（1分钟）”进入向导。
3. 可点击“打开设置”进入完整配置页。

> 截图位置：`S01-02`  
> 建议截图：首次启动时的“模型快速配置 Gate/提示条”。

完成快速配置后，再确认以下主界面元素：
1. 左侧能看到“开始任务 / 专家技能 / 智能体员工”。
2. 左下角能看到“设置”按钮。
3. 页面无明显空白或错误提示。

> 截图位置：`S01-03`  
> 建议截图：完成配置后的主界面全屏，标注左侧导航和设置按钮。

## 常见问题
1. 打不开应用：重启系统后再次尝试启动。
2. 启动后白屏：先关闭重开；仍异常时提交截图给维护同学。
3. 启动后无法点击：等待 10-20 秒再试，避免同时多次启动。
